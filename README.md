# Saiteja Desu - Portfolio Website

A minimal, clean Jekyll portfolio website built for GitHub Pages.

## Included Files

- `index.md` - Single-page portfolio content (easy to edit sections)
- `_config.yml` - Jekyll and GitHub Pages configuration
- `assets/css/style.css` - Custom styling and responsive layout
- `README.md` - Setup and editing guide

## Run Locally

1. Install Ruby and Bundler.
2. Install Jekyll:
   ```bash
   gem install jekyll bundler
   ```
3. Serve the site:
   ```bash
   jekyll serve
   ```
4. Open `http://localhost:4000`.

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings -> Pages**.
3. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (root)
4. Save. GitHub Pages will build and publish your site.

## Quick Content Editing

Open `index.md` and update:
- Name + headline in the hero section
- About paragraph from your latest LinkedIn bio
- Skills chips
- Projects (replace placeholders with real links and outcomes)
- Experience bullets with quantified impact
- Awards and contact details

## Notes

- Uses the GitHub Pages-supported `minima` theme.
- No extra plugins are required.
- Custom styles are in `assets/css/style.css`.
